Swift Package Manager

The Swift Package Manager, or SPM, provides a way to create Swift libraries and Swift apps on macOS and Linux.

SPM also provides a way for you to manage your dependencies and makes sharing Swift code easier.

Kitura, and all of it's packages are written in Swift which is made usuable by SPM, that is you will use SPM to add Kitura to a Swift project.

Add an SPM Package to your Swift project

SPM has GitHub integration so you can use the URL of a GitHub repo as the source.
