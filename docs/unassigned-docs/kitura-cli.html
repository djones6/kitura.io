Kitura CLI

Prerequisites

You need to have the Kitura CLI installed.

Using the CLI

There are various commands you can run via the CLI.

Command Description
kitura init Create a cloud ready Kitura application.
kitura api Create a cloud ready Kitura application with OpenAPI for testing.
kitura basic Create a bare bones Kitura project with a `Hello World` example route.
kitura create Creare a custom Kitura project via command prompts.
kitura idt Install IBM Cloud Developer Tools
kitura build Build the project in a local container using IBM Cloud Developer Tools.
kitura run Run the project in a local container using IBM Cloud Developer Tools.
kitura --help Display information about each Kitura CLI command.
kitura help [cmd] Display information about a specific Kitura CLI command.

Using the Kitura CLI

The Kitura Command Line Interface, or CLI, provides easy to use commands for creating a Kitura server.

NOTE: Kitura CLI commands need to be run in an empty directory
