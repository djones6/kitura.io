Kitura Redis

Prerequisites

To test Kitura-Redis locally you need to install Redis:

brew install redis

Once Redis is installed you need to start the service.

There's two options for starting the Redis service:

Start Redis as a background service:

brew services start redis

Start a single non-background instance:

redis-server /usr/local/etc/redis.conf
