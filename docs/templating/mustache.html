Step 1: Add Kitura-Mustache to your project

In your Package.swift you need to add the following:

.package(url: "https://github.com/IBM-Swift/Kitura-MustacheTemplateEngine.git", from: "x.x.x"),

Where x.x.x represents the latest version

For more information on adding Kitura-Mustache to your project you can view the Readme.

Finally you will need to add Kitura-Mustache to your build targets.

Step 2: Build your project

Since you've added a new package to your Package.swift you need to rebuild your project.

In a terminal run the following:

swift package update

If you're on macOS and using Xcode you will also need to generate a new xcode project:

swift package generate-xcodeproj

Step 3: Create template files

For Kitura to make use of templating you need template files to work with.

By default Kitura template engines look in a ./Views directory for the template files.

In the root of your project, run the following in a terminal:

mkdir Views

Note: If you're working in Xcode you will need to generate a new Xcode project:

swift package generate-xcodeproj

Step 3: Add Kitura-Mustache to your App code

Import KituraMustache into your project

import KituraMustache

Add KituraMustache to your router:

router.add(templateEngine: MustacheTemplateEngine())

Step 4: Render a template


      router.get("/hello") { request, response, next in
        try response.render("Example.mustache", context: ["name": "Kitura!"])
      }
